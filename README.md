## Running the Project ##

### Backend ###
1. Navigate to the `Backend` directory:
    ```bash
    cd Backend
    ```
2. To start the backend normally:
    ```bash
    npm run start
    ```
3. To start the backend with nodemon for development:
    ```bash
    npm run devStart
    ```

### Frontend ###
1. Navigate to the `cute-cumbers` directory:
    ```bash
    cd cute-cumbers
    ```
2. Start the frontend:
    ```bash
    npm start
    ```

## Database Information ##
